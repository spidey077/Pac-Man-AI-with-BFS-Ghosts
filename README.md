Pac-Man AI with BFS Ghosts
This is a modern twist on the classic Pac-Man game. Unlike the original, where ghosts follow simple pre-defined patterns, this version uses AI (Breadth-First Search) to make ghosts intelligent and actively chase the player.

Features:
Classic Pac-Man gameplay with maze navigation.
AI-controlled ghosts using BFS pathfinding for smarter, more challenging behavior.
Collectible pellets with scoring system.
Multiple lives with a pause and interactive pop-up on life lost.
Clean UI displaying score, high score, lives, time, and remaining pellets.
Sound effects and background music support.

Technologies Used:
Python 3
Pygame

How to Play:
Use arrow keys to move Pac-Man.
Avoid ghosts; collect all pellets to win.
If you lose a life, press Enter to continue.
